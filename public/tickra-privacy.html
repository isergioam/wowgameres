Política de Privacidad

Última actualización: 17 de junio de 2026

WOWGamerES ("nosotros", "nuestro" o "la Empresa") se compromete a proteger la privacidad de los usuarios de la aplicación móvil Tickra ("la Aplicación"). Esta Política de Privacidad describe cómo se recopila, utiliza y protege tu información al utilizar nuestros servicios.

1. Información que recopilamos

  • Datos guardados localmente: Toda la información de tus compras, artículos extraídos, totales, tiendas, fechas y categorías de gastos se almacena exclusivamente en la base de datos local de tu dispositivo móvil (base de datos SQLite/Room). Nosotros no tenemos acceso a este historial de datos ni lo guardamos en servidores externos.
  • Imágenes de tickets: Cuando utilizas la función de cámara para escanear un ticket, la imagen se procesa temporalmente con el único propósito de extraer la información de compra. La imagen se transmite de forma cifrada a través de nuestra API segura a los servicios de procesamiento de Inteligencia Artificial (Google Gemini API). Estas imágenes no se almacenan de forma permanente en ningún servidor y se eliminan inmediatamente tras finalizar el proceso de extracción de datos.

2. Servicios de Terceros

La Aplicación hace uso de servicios de terceros para el análisis inteligente de imágenes y la detección de textos:

  • Google Gemini API: Las imágenes de los tickets se envían temporalmente a esta API para convertirlas en datos de texto estructurados. Google procesa estos datos de conformidad con sus políticas de privacidad aplicables para desarrolladores de API.

3. Uso de la Información

Utilizamos la información recopilada estrictamente para:

  • Proporcionar la funcionalidad central de la aplicación (conversión de imágenes de tickets a listas de artículos de gastos).
  • Mantener estadísticas de gastos financieros organizados dentro de la aplicación móvil.
  • Permitir la exportación de tus datos a formato de hoja de cálculo Excel.

4. Seguridad de los Datos

Implementamos medidas técnicas y organizativas de seguridad estándar de la industria para proteger tus datos contra el acceso no autorizado, la alteración o la divulgación, especialmente durante la transferencia de las imágenes a la API de análisis.

5. Derechos de Control del Usuario

Al estar todos los datos financieros almacenados de forma local en tu dispositivo móvil, tienes control total sobre ellos:

  • Puedes eliminar registros individuales de tickets o artículos en cualquier momento desde la propia interfaz de la aplicación.
  • Si desinstalas la aplicación o borras sus datos desde los ajustes del sistema de Android, toda tu base de datos de compras y configuraciones se eliminará de forma permanente e irrecuperable.
